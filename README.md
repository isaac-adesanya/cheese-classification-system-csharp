# Cheese Classification System

A C# WPF desktop application designed to classify, store, update, and manage different types of cheeses using a SQLite database.

## Technologies Used
- C#
- SQLite
- SQL
- Visual Studio

## Features
- Add, update, delete, and view cheese records
- Store cheese types, batches, and related information in a database
- Full CRUD (Create, Read, Update, Delete) functionality
- Input validation and error handling

## What I Learned
- Building structured desktop applications in C#
- Integrating SQLite databases using SQL queries
- Implementing CRUD operations
- Applying object-oriented programming principles
